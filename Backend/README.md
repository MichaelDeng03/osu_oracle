### Environment Variables

To configure OAuth2 with this application, create a `.env` file in the `Backend/` directory with the following variables:
Please refer to the official [osu! API documentation](osu.ppy.sh/docs/#registering-an-oauth-application)

```
OSU_CLIENT_ID=your_osu_client_id
OSU_CLIENT_SECRET=your_osu_client_secret
```
